# Product Images

Place your product images in this directory with the following filenames:

- `kimchi.jpg` - Kimchi Original product image
- `ricecoffee.jpg` - Rice Coffee product image
- `radish.jpg` - Pickled Radish product image
- `kimchi-spicy.jpg` - Kimchi Spicy product image
- `ricecoffee-premium.jpg` - Rice Coffee Premium product image
- `pickled-set.jpg` - Pickled Vegetables Set product image

**Recommended image specifications:**
- Format: JPG or PNG
- Size: 800x800px (square) or 1200x1200px for better quality
- Optimize images for web (reduce file size for faster loading)

The website will display placeholder gradients if images are not found, so the site will work even without images initially.
